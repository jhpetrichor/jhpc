Usage: ./Core PPI-file output-file
	e.g. ./Core DIP.txt DIP-Complex.out

PPI-file: 
	- Each line describing an Protein-Protein Interaction
	- Each line contain two proteins which are seperated by blanks

output-file:
	- Each line describe a predicted complex
